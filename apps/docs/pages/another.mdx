# Getting Started

import { Callout } from "nextra-theme-docs";

Use our [Pre-Configured Template Repository](https://github.com/Palanikannan1437/RC4-Starter-Pack) to get started ⚡️

<Callout type="info" emoji="ℹ️">
  Use the [`app dir`](https://beta.nextjs.org/docs/installation) of Next.js 13
</Callout>

1. Install @palanikannan1437/rc4-ui from npm
2. Install @palanikannan1437/rc4-core from npm

_*rc4-core would be considered to be merged in the future with rc4-ui*_

```
pnpm i @palanikannan1437/rc4-ui
pnpm i @palanikannan1437/rc4-core
```

4. Get Started with the [Timeline Component](./components/Timeline)
